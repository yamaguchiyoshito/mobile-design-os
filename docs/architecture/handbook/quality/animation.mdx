🔴 **MUST NOT**: `Animated`（React Native コア）を新規実装に使わない — JSスレッド駆動でフレームドロップを引き起こす  
🔴 **MUST NOT**: `PanResponder` を新規実装に使わない — 同上  
🔴 **MUST**: `GestureHandlerRootView` を `app/_layout.tsx` の最外周に配置する（01_Principles で定義済み）

### `useReducedMotion` の使い方

Reanimated v3 には `useReducedMotion` が組み込まれているため、**独自フックは作成しない**。

```ts
// ✅ Reanimated 公式フックを直接使う
import { useReducedMotion } from 'react-native-reanimated';

function MyComponent() {
  const reduceMotion = useReducedMotion();
  // ...
}
```

> **なぜカスタムフックを廃止するか**: Reanimated の worklet 最適化と連動しており、カスタム実装では `useAnimatedStyle` 内での参照が正しく機能しないケースがある。

🔴 **MUST**: すべてのアニメーションで `useReducedMotion` を確認する  
🟡 **SHOULD**: スライド・バウンスはモーション制限時にフェードへ置き換える。フェードは「変化の知覚」を保持しながら動きを最小化できる唯一の代替手段

## 2. 実装層：Reanimated の基本パターン

### shared value と animated style

```tsx
import Animated, {
  useSharedValue, useAnimatedStyle, withSpring, withTiming,
} from 'react-native-reanimated';

export function FadeInCard({ children }: { children: React.ReactNode }) {
  const opacity    = useSharedValue(0);
  const translateY = useSharedValue(16);

  useEffect(() => {
    // shared value は参照が変わらないため依存配列に含めなくてよい
    // eslint-disable-next-line react-hooks/exhaustive-deps
    opacity.value    = withTiming(1,  { duration: 200 });
    translateY.value = withSpring(0,  { damping: 20, stiffness: 300 });
  }, []);

  const animatedStyle = useAnimatedStyle(() => ({
    opacity:   opacity.value,
    transform: [{ translateY: translateY.value }],
  }));

  return (
    <Animated.View style={animatedStyle}>
      {children}
    </Animated.View>
  );
}
```

### `withSequence` / `withDelay` の使い方

複数アニメーションを時間軸で制御する場合に使う。

```tsx
import { withSequence, withDelay, withTiming, withSpring } from 'react-native-reanimated';

// ✅ 連続実行: 押下エフェクト → 元に戻る
scale.value = withSequence(
  withTiming(0.9, { duration: 80 }),
  withSpring(1,   { damping: 15 }),
);

// ✅ 遅延付き: ステガー表示（アイテムを順番に表示する）
opacity.value = withDelay(index * 50, withTiming(1, { duration: 200 }));

// ✅ 組み合わせ: 500ms 後に開始し、その後 spring で戻る
translateY.value = withDelay(
  500,
  withSequence(
    withTiming(-10, { duration: 150 }),
    withSpring(0,   { damping: 20 }),
  ),
);
```

### `interpolate` と `Extrapolation` の選択基準

```tsx
import { interpolate, Extrapolation } from 'react-native-reanimated';

// CLAMP: 入力範囲の外では出力を端値で固定する（最も一般的）
// → スクロール連動でヘッダーが "それ以上縮まない" ような制限に使う
const height = interpolate(
  scrollOffset.value,
  [0, 140],
  [200, 60],
  Extrapolation.CLAMP,  // scrollOffset < 0 でも height は 200 のまま
);

// EXTEND: 入力範囲の外でも線形に延長する
// → ゴム紐効果（引っ張るほど動く）など意図的なオーバーシュートに使う
const rubberBand = interpolate(
  dragY.value,
  [0, 100],
  [0, 30],
  Extrapolation.EXTEND,
);

// IDENTITY: 範囲外では出力 = 入力をそのまま返す（ほぼ使わない）
// → 入力値と出力値の単位が同じで、変換が不要な範囲のみに留めたい特殊ケース
```

## 3. レイアウトアニメーション

要素の出現・消去・高さ変化に `entering` / `exiting` / `layout` を使う。

```tsx
import Animated, {
  FadeIn, FadeOut, SlideInDown, SlideOutDown,
  LinearTransition,
} from 'react-native-reanimated';
import { useReducedMotion } from 'react-native-reanimated';

// ✅ アイテムの追加・削除アニメーション
export function AnimatedList({ items }: { items: Item[] }) {
  return (
    <View>
      {items.map((item) => (
        <Animated.View
          key={item.id}
          entering={FadeIn.duration(200)}
          exiting={FadeOut.duration(150)}
          layout={LinearTransition}
        >
          <ItemCard item={item} />
        </Animated.View>
      ))}
    </View>
  );
}

// ✅ モーション制限に対応したカード
export function AnimatedCard({ children }: { children: React.ReactNode }) {
  const reduceMotion = useReducedMotion();

  const entering = reduceMotion
    ? FadeIn.duration(100)
    : SlideInDown.springify().damping(20);

  const exiting = reduceMotion
    ? FadeOut.duration(100)
    : SlideOutDown.duration(200);

  return (
    <Animated.View entering={entering} exiting={exiting}>
      {children}
    </Animated.View>
  );
}
```

## 4. ジェスチャーハンドラ

### Tap（ボタン代替）

```tsx
import { Gesture, GestureDetector } from 'react-native-gesture-handler';
import Animated, {
  useSharedValue, useAnimatedStyle, withSpring, runOnJS,
} from 'react-native-reanimated';

export function PressableCard({ onPress, children }: PressableCardProps) {
  const scale = useSharedValue(1);

  const tap = Gesture.Tap()
    .onBegin(()   => { scale.value = withSpring(0.96, { damping: 20 }); })
    .onFinalize(() => { scale.value = withSpring(1,    { damping: 20 }); })
    // GestureコールバックはUIスレッド（worklet）で実行されるため、
    // JSスレッドの関数（onPress）は runOnJS でラップして呼び出す
    .onEnd(() => { runOnJS(onPress)(); });

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ scale: scale.value }],
  }));

  return (
    <GestureDetector gesture={tap}>
      <Animated.View
        style={animatedStyle}
        accessible={true}
        accessibilityRole="button"
      >
        {children}
      </Animated.View>
    </GestureDetector>
  );
}
```

### LongPress（長押し）

```tsx
import { Gesture, GestureDetector } from 'react-native-gesture-handler';
import Animated, {
  useSharedValue, useAnimatedStyle, withTiming, withSpring, runOnJS,
} from 'react-native-reanimated';

const LONG_PRESS_DURATION = 500; // ms

export function LongPressCard({ onLongPress, children }: LongPressCardProps) {
  const scale   = useSharedValue(1);
  const opacity = useSharedValue(1);

  const longPress = Gesture.LongPress()
    .minDuration(LONG_PRESS_DURATION)
    .onBegin(() => {
      // 押し込みエフェクト
      scale.value   = withTiming(0.95, { duration: LONG_PRESS_DURATION });
      opacity.value = withTiming(0.8,  { duration: LONG_PRESS_DURATION });
    })
    .onStart(() => {
      runOnJS(onLongPress)();
    })
    .onFinalize(() => {
      scale.value   = withSpring(1,   { damping: 20 });
      opacity.value = withTiming(1,   { duration: 150 });
    });

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ scale: scale.value }],
    opacity:   opacity.value,
  }));

  return (
    <GestureDetector gesture={longPress}>
      <Animated.View
        style={animatedStyle}
        accessible={true}
        accessibilityRole="button"
        accessibilityHint="長押しでメニューを表示"
      >
        {children}
      </Animated.View>
    </GestureDetector>
  );
}
```

### Pan（ドラッグ・スワイプ）

```tsx
import { Gesture, GestureDetector } from 'react-native-gesture-handler';
import Animated, {
  useSharedValue, useAnimatedStyle,
  withSpring, withTiming, runOnJS,
} from 'react-native-reanimated';

const DISMISS_THRESHOLD = 120; // px。design-tokens/animation.ts で管理

export function SwipeToDismissCard({ onDismiss, children }: SwipeProps) {
  const translateX = useSharedValue(0);
  const opacity    = useSharedValue(1);

  const pan = Gesture.Pan()
    .onUpdate((e) => {
      translateX.value = e.translationX;
      opacity.value    = 1 - Math.abs(e.translationX) / 300;
    })
    .onEnd((e) => {
      if (Math.abs(e.translationX) > DISMISS_THRESHOLD) {
        // 閾値超え → 画面外へ飛ばしてから dismiss を呼ぶ
        // translateX と opacity を同時に開始し、完了コールバックは translateX 側のみ
        const target = e.translationX > 0 ? 500 : -500;
        opacity.value    = withTiming(0, { duration: 200 });
        translateX.value = withTiming(
          target,
          { duration: 200 },
          (finished) => { if (finished) runOnJS(onDismiss)(); },
        );
      } else {
        // 閾値未満 → 同時に元の位置へ戻す
        translateX.value = withSpring(0, { damping: 20 });
        opacity.value    = withSpring(1, { damping: 20 });
      }
    });

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ translateX: translateX.value }],
    opacity:   opacity.value,
  }));

  return (
    <GestureDetector gesture={pan}>
      <Animated.View style={animatedStyle}>{children}</Animated.View>
    </GestureDetector>
  );
}
```

### ジェスチャーの競合解決

```tsx
// スクロールとスワイプが競合する場合
const pan    = Gesture.Pan().activeOffsetX([-10, 10]); // X 方向 10px 以上でアクティブ
const scroll = Gesture.Native();                        // ScrollView / FlatList のネイティブジェスチャー

// 同時認識（どちらも生きている）
const composed  = Gesture.Simultaneous(scroll, pan);

// pan が成立したら scroll をキャンセル（水平スワイプ専用のリストに使う）
const exclusive = Gesture.Exclusive(pan, scroll);
```

🟡 **SHOULD**: `ScrollView` / `FlatList` 内でドラッグジェスチャーを使う場合は `Simultaneous` か `Exclusive` で競合を明示的に解決する

## 5. worklet ルール

### 呼び出しコンテキストの分類

```tsx
// ✅ worklet 内で使えるもの
useAnimatedStyle(() => {
  return {
    opacity:   opacity.value,           // shared value の読み取り
    transform: [{ scale: scale.value }], // 四則演算・三角関数
  };
});

// ❌ worklet 内で直接使えないもの（runOnJS / runOnUI でラップする）
useAnimatedStyle(() => {
  console.log(opacity.value); // ❌ UIスレッドに console はない
  setState(opacity.value);    // ❌ setState は JSスレッドの操作
  router.push('/next');       // ❌ ナビゲーションは JSスレッドの操作
});
```

### `runOnJS` / `runOnUI` の使い分け

```tsx
// runOnJS: UIスレッド（worklet） → JSスレッドへの呼び出し
pan.onEnd(() => {
  runOnJS(router.push)('/next-screen'); // ナビゲーション
  runOnJS(setIsOpen)(false);            // setState
  runOnJS(onDismiss)();                 // コールバックprop
});

// runOnUI: JSスレッド → UIスレッドへの呼び出し
// 通常は worklet 内で自動変換されるため使用頻度は低い
// useEffect 内から shared value を操作する場合など
runOnUI(() => {
  'worklet';
  opacity.value = withTiming(0);
})();
```

## 6. 共通アニメーションコンポーネント

繰り返し使うパターンは Design System に共通化する。

### アニメーション定数の管理場所

```ts
// src/constants/animation.ts に一元管理する
export const ANIMATION = {
  DISMISS_THRESHOLD:   120,   // スワイプ dismiss の閾値 (px)
  SPRING_DAMPING:       20,   // 共通 spring のダンピング係数
  SPRING_STIFFNESS:    300,   // 共通 spring の剛性
  FADE_DURATION:       200,   // 標準フェード時間 (ms)
  SLIDE_DURATION:      300,   // 標準スライド時間 (ms)
  SKELETON_DURATION:   700,   // スケルトンパルス時間 (ms)
  LONG_PRESS_DURATION: 500,   // 長押し認識時間 (ms)
} as const;
```

### FadeView

```tsx
// src/components/ui/Animated/FadeView.tsx
import { useReducedMotion } from 'react-native-reanimated';
import { ANIMATION } from '@/constants/animation';

type FadeViewProps = {
  visible:   boolean;
  duration?: number;
  children:  React.ReactNode;
  style?:    ViewStyle;
};

export function FadeView({ visible, duration = ANIMATION.FADE_DURATION, children, style }: FadeViewProps) {
  const reduceMotion = useReducedMotion();
  const opacity      = useSharedValue(visible ? 1 : 0);

  useEffect(() => {
    opacity.value = withTiming(visible ? 1 : 0, {
      duration: reduceMotion ? 0 : duration,
    });
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [visible, duration, reduceMotion]);

  const animatedStyle = useAnimatedStyle(() => ({
    opacity: opacity.value,
  }));

  return (
    <Animated.View style={[animatedStyle, style]}>
      {children}
    </Animated.View>
  );
}
```

### SkeletonPulse

```tsx
// src/components/ui/Animated/SkeletonPulse.tsx
import { useReducedMotion } from 'react-native-reanimated';
import { ANIMATION } from '@/constants/animation';

export function SkeletonPulse({ style }: { style?: ViewStyle }) {
  const reduceMotion = useReducedMotion();
  const opacity      = useSharedValue(1);

  useEffect(() => {
    if (reduceMotion) return;

    opacity.value = withRepeat(
      withTiming(0.3, {
        duration: ANIMATION.SKELETON_DURATION,
        easing:   Easing.inOut(Easing.ease),
      }),
      -1,   // 無限ループ
      true, // reverse
    );

    // ⚠️ アンマウント時に必ずキャンセルする（メモリリーク防止）
    return () => {
      cancelAnimation(opacity);
    };
  }, [reduceMotion]);

  const animatedStyle = useAnimatedStyle(() => ({
    opacity: opacity.value,
  }));

  return (
    <Animated.View
      style={[{ backgroundColor: colors['surface-secondary'] }, animatedStyle, style]}
      accessible={true}
      accessibilityLabel="読み込み中"
      accessibilityRole="progressbar"
    />
  );
}
```

## 7. スクロール連動アニメーション

```tsx
import Animated, {
  useAnimatedRef, useScrollViewOffset,
  useAnimatedStyle, interpolate, Extrapolation,
} from 'react-native-reanimated';

const HEADER_MAX_HEIGHT = 200;
const HEADER_MIN_HEIGHT = 60;

export function AnimatedHeaderScreen() {
  const scrollRef    = useAnimatedRef<Animated.ScrollView>();
  const scrollOffset = useScrollViewOffset(scrollRef);

  const headerStyle = useAnimatedStyle(() => {
    const height = interpolate(
      scrollOffset.value,
      [0, HEADER_MAX_HEIGHT - HEADER_MIN_HEIGHT],
      [HEADER_MAX_HEIGHT, HEADER_MIN_HEIGHT],
      Extrapolation.CLAMP, // マイナスオフセット（バウンス）でもヘッダーが広がらないようにする
    );
    return { height };
  });

  const titleOpacity = useAnimatedStyle(() => ({
    opacity: interpolate(
      scrollOffset.value,
      [0, 50, 100],
      [1, 0.5, 0],
      Extrapolation.CLAMP,
    ),
  }));

  return (
    <View style={{ flex: 1 }}>
      <Animated.View style={[styles.header, headerStyle]}>
        <Animated.Text style={[styles.title, titleOpacity]}>
          画面タイトル
        </Animated.Text>
      </Animated.View>
      <Animated.ScrollView ref={scrollRef}>
        {/* コンテンツ */}
      </Animated.ScrollView>
    </View>
  );
}
```

🟡 **SHOULD**: スクロール連動アニメーションは `useScrollViewOffset` を使う（`onScroll` を使うとJSスレッドを介するためフレームドロップが発生する）

## 8. テスト

### モック設定の方針

Reanimated のモックは **`jest.config.ts` のグローバル設定のみ** で管理する。個々のテストファイルで `jest.mock('react-native-reanimated', ...)` を重複定義しない。

```ts
// jest.config.ts
export default {
  preset: 'react-native',
  setupFilesAfterFramework: ['@testing-library/react-native/extend-expect'],
  transformIgnorePatterns: [
    'node_modules/(?!(react-native|react-native-reanimated|react-native-gesture-handler)/)',
  ],
  moduleNameMapper: {
    'react-native-reanimated': 'react-native-reanimated/mock',
  },
};
```

### アニメーションコンポーネントのテスト方針

Reanimated v3 のモックはアニメーション値を即時解決するため、**最終状態のみ検証する**。中間フレームのテストは不要かつ不安定。

```tsx
// src/components/ui/Animated/__tests__/FadeView.test.tsx
import { render } from '@testing-library/react-native';
import { FadeView } from '../FadeView';

// グローバルモックが効いているため jest.mock の重複記述は不要

test('visible=false のとき opacity が 0 になる', () => {
  const { getByTestId, rerender } = render(
    <FadeView visible={true} testID="fade">
      <Text>コンテンツ</Text>
    </FadeView>,
  );

  rerender(
    <FadeView visible={false} testID="fade">
      <Text>コンテンツ</Text>
    </FadeView>,
  );

  expect(getByTestId('fade').props.style).toMatchObject({ opacity: 0 });
});
```

### スナップショットテストの方針

アニメーションコンポーネントのスナップショットは **静止状態（初期値）のみ** 記録する。アニメーション中の中間状態をスナップショットに含めると、Reanimated のバージョンアップでノイズが発生する。

```tsx
test('FadeView の初期スナップショット（visible=true）', () => {
  const { toJSON } = render(
    <FadeView visible={true}>
      <Text>コンテンツ</Text>
    </FadeView>,
  );
  expect(toJSON()).toMatchSnapshot();
});
```
